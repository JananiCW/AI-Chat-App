<h1 align = "center"> 🤖 AI Chat App </h1>

A full-stack AI-powered chat application built with **React**, **Go (Gin)**, and **MongoDB**, using **Google Gemini AI**.

---

## 🌟 Features

- 💬 Real-time AI chat powered by Google Gemini
- 🗂️ Multiple chat sessions
- 📝 Chat history saved to MongoDB
- 🗑️ Delete conversations
- ⚡ Typing animation while AI responds
- 🌙 Beautiful dark theme UI

---

## 🧱 Tech Stack

| Part       | Technology              |
|------------|-------------------------|
| Frontend   | React                   |
| Backend    | Go + Gin                |
| AI         | Google Gemini API       |
| Database   | MongoDB (Local)         |
